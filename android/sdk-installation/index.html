SDK Installation

SDK Installation

To properly install Paylivre's SDK Gateway follow the steps below.

Add dependencies

We recommend using Android Studio as an IDE and Gradle to manage dependencies.
You will need to add 2 dependencies: (step 1) jitpack*; and (step 2) our implementation.
* For the jitpack, depending on your project organization, you will have 2 different scenarios (both shown below).

Step 1: jitpack - Scenario A (older projects)

  • To your root build.gradle at the end of repositories, please add the following dependencies:
./build.gradle
1// ... other configs
2allprojects {
3    repositories {
4      // ... other repositories
5      maven { url 'https://jitpack.io' }
6    }
7}

Step 1: jitpack - Scenario B (newer projects)

  • To your root settings.gradle inside of dependencyResolutionManagement > repositories, please add the following dependencies:
./settings.gradle
1// ... other configs
2dependencyResolutionManagement {
3  // ... other dependencies
4  repositories {
5    // ... other repositories
6    maven { url 'https://jitpack.io' }
7  }
8}

Step 2: Our implementation

  • For the packages that you chose to implement Paylivre's SDK Gateway (example - 'app'), open the build.gradle and add the following implementation:
./app/build.gradle
1// ... other configs
2dependencies {
3  // ... other dependencies
4  implementation 'com.github.neijrdev:sdk-gateway-android:1.0.1'
5}
