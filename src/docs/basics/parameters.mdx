## Parameters table

| Key                     | Required | Type   | Description                                                                                                                                                                                                       |
| ----------------------- | -------- | ------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| merchant_id             | Yes      | Int    | Your Account ID at Paylivre                                                                                                                                                                                       |
| gateway_token           | Yes      | String | ...                                                                                                                                                                                                               |
| operation               | Yes      | String | The request’s operation type. 0 = Deposit/Cash-In or 5 = Withdrawal/Cash-Out.                                                                                                                                     |
| merchant_transaction_id | Yes      | String | The request’s unique transaction identification in your system                                                                                                                                                    |
| amount                  | Yes      | Int    | The request’s amount. Amount multiplied by 100 to remove decimals. Minimum amount is 500 regardless of currency.                                                                                                  |
| currency                | Yes      | String | The request’s currency. Supported: BRL or USD                                                                                                                                                                     |
| type                    | Yes      | Int    | Represents the bitmask with the available payment method options.                                                                                                                                                 |
| account_id              | Yes      | String | The customer’s unique user identification in your system                                                                                                                                                          |
| callback_url            | Yes      | String | URL to receive callbacks from                                                                                                                                                                                     |
| base_url                | Yes      | String | https://playground.gateway.paylivre.com/ (for playground environment) or https://web.gateway.paylivre.com/ (for production environment)                                                                           |
| email_address           | No       | String | The requesting customer’s email address.                                                                                                                                                                          |
| document                | No       | String | The requesting customer’s tax document id (CPF or CNPJ).                                                                                                                                                          |
| auto_approve            | Yes      | Int    | 0 = false or 1 = true. When true, it automatically queues the withdrawal for processing, skipping the need of your manual approval in Paylivre’s backoffice. (Only utilized for Withdrawal/Cash-Out Transactions) |
| logo_url                | No       | String | Your logo's url (image) to be displayed in the checkout page                                                                                                                                                      |
| pix_key_type            | No       | Int    | The type of the customer’s PIX Key. Mandatory when parameter pix_key is used, otherwise optional. (0 = Tax ID document Number, 2 = Telephone Number or 3 = Email Address)                                         |
| pix_key                 | No       | String | Only necessary for Withdrawal Pix                                                                                                                                                                                 |
