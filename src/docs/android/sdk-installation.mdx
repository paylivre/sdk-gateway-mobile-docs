To properly install Paylivre's SDK Gateway for Android projects follow the steps below.

## Add dependencies

We recommend using Android Studio as an IDE and Gradle to manage dependencies.<br/>
You will need to add 2 dependencies: (step 1) jitpack\*; and (step 2) our implementation.<br /> \* For the jitpack, depending on your project organization, you will have 2 different
scenarios, both shown below:

### Step 1: jitpack - Scenario A (newer projects)

To your root `settings.gradle`, inside of `dependencyResolutionManagement` > `repositories`, please add the following dependencies:

```groovy title=./settings.gradle lineNumbers=true highlight={6}
// ... other configs
dependencyResolutionManagement {
  // ... other dependencies
  repositories {
    // ... other repositories
    maven { url 'https://jitpack.io' }
  }
}
```

### Step 1: jitpack - Scenario B (older projects)

To your root `build.gradle`, at the end of `repositories`, please add the following dependencies:

```groovy title=./build.gradle lineNumbers=true highlight={5}
// ... other configs
allprojects {
    repositories {
      // ... other repositories
      maven { url 'https://jitpack.io' }
    }
}
```

### Step 2: Our implementation

For the packages that you choose to implement Paylivre's SDK Gateway (example: 'app'), open its `build.gradle` and add the following implementation:

```groovy title=./app/build.gradle lineNumbers=true highlight={4}
// ... other configs
dependencies {
  // ... other dependencies
  implementation 'com.github.neijrdev:sdk-gateway-android:1.0.1'
}
```

Done!
