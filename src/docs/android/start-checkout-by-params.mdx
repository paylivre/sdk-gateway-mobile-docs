Implementing with `StartCheckoutByParams` class consists as the alternative to the <Link to="/android/start-checkout-by-url">StartCheckoutByURL</Link> for the merchant to integrate with Paylivre's SDK Gateway. It is suggested when the merchant do not want to handle the generation of the URL by itself, thus, it is required to provide the parameters.<br/>
In your app, you can invoke our SDK at any given momment, being at the `Activity` or at any given `fragment`.<br/> The examples in this section below will occour at the `MainActivity.kt`.

## Implementation

StartCheckoutByParams's `Builder()`, accepts all the <Link to="/basics/parameters">parameters listed in the BASICS-Parameters</Link> section.

We recommend you instatiating the `StartCheckoutByParams` class inside of a new function called `startCheckoutByParams`, as follows:

### Kotlin implementation

```kotlin title=./java/MainActivity.kt lineNumbers=true highlight={6-32}
class MainActivity : AppCompatActivity() {
  override fun onCreate(savedInstanceState: Bundle?) {
    // ...other

    // new function declared
    fun startCheckoutByParams() {
        // This variable is an identifier that can be pass optionally to the Builder(). If so, i will be used for the method onActivityResult
        // val SDK_GATEWAY_PAYLIVRE_ACTIVITY_REQUEST_CODE = 1

        val checkout = StartCheckoutByParams.Builder(
            // request_code = SDK_GATEWAY_PAYLIVRE_ACTIVITY_REQUEST_CODE, // Optional
            merchant_id = 302,
            gateway_token = "NHsuzedl6omTPvoxc0p7gVXc7Xthhf6Y",
            operation = 0, // 0 deposit OR 5 withdrawal
            merchant_transaction_id = "Okosdksod2301",
            amount = 500,
            currency = "USD", // Suportted: BRL or USD
            type = 1, // see type index table
            account_id = "AccIdTest",
            callback_url = "https://www.google.com",
            base_url = "https://dev.gateway.paylivre.com",
            email_address = "user_gateway_test@tutanota.com", // Optional
            document = "61317581075", // Optional
            auto_approve = 0, // 0 = false or 1 = true
            logo_url = "https://github.com/paylivre/gateway-example-react-js/blob/master/assets/logo_jackpot_new.png?raw=true",
            pix_key_type = 0, // Optional
            pix_key = "61317581075" // Optional
        ).build()

      // method that initializes the SDK
      checkout.startPayment(this)
    }
    startCheckoutByParams() //  run the function
  }
}
```

## Extra addons

As extra addons, Paylivre's SDK Gateway Mobile offers three possibilities, through three different methods:

- Define a <Link to="/extras/language">language\*</Link> using the `setLanguage()` method;
- Define your <Link to="/extras/logo-image">merchant's logo image from a file\*</Link> using the `setLogoResId()` method;
- Register callback for <Link to="/extras/activity-result">activity result\*</Link> using the `setRegisterForResult()` method.<br/>
  \*Refer to the links to learn more about each of them.

In case you want to implement any of them, following the same structure presented in the [implementation above](#implementation), just add these lines of code before your `startPayment()` method.

### Kotlin extras

```kotlin title=./java/MainActivity.kt lineNumbers=true highlight={12,15-16,19}
class MainActivity : AppCompatActivity() {
  override fun onCreate(savedInstanceState: Bundle?) {
    // ...other

    // new function declared
    fun startCheckoutByParams() {
      val checkout = StartCheckoutByParams.Builder(
          // ...all the parameters
      ).build()

      // set language
      checkout.setLanguage("en-EN")

      // set merchant's logo
      val MERCHANT_LOGO_RES_ID: Int = R.drawable.logo_merchant // loading from a given file
      checkout.setLogoResId(MERCHANT_LOGO_RES_ID)

      // Register callback for activity result
      checkout.setRegisterForResult(checkoutRegisterForResult)

      // method that initializes the SDK
      checkout.startPayment(this)
    }
    startCheckoutByParams() //  run the function
  }
}
```
